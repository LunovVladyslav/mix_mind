# Delay Theory & Usage

## BPM-Synced Delay Times (ms)
Formula: 60000 / BPM = quarter note ms

| BPM | 1/4 note | 1/8 note | dotted 1/8 | 1/16 note |
|-----|---------|---------|-----------|----------|
| 80  | 750ms   | 375ms   | 562ms     | 187ms    |
| 100 | 600ms   | 300ms   | 450ms     | 150ms    |
| 120 | 500ms   | 250ms   | 375ms     | 125ms    |
| 128 | 469ms   | 234ms   | 352ms     | 117ms    |
| 140 | 429ms   | 214ms   | 321ms     | 107ms    |
| 160 | 375ms   | 188ms   | 281ms     | 94ms     |

## When to Use Delay
- Vocal doublers: short delay 20-80ms (slapback)
- Adding width: stereo delay L/R with different times
- Rhythmic interest: dotted 1/8 note delay is classic
- Spatial depth: longer delays for background elements

## Delay Types by Genre
- Pop/Rock: dotted 1/8 note, synced, 20-30% feedback
- Dub/Reggae: long delays, high feedback, tape echo character
- Country: slapback echo 60-120ms, single repeat
- Electronic: ping-pong stereo delay, synced to grid
- Ambient: long delays with high feedback and filter

## Where to Place Delay
- SEND (preferred): keeps dry signal clean, control wet amount per channel
- INSERT only for: slapback vocals, tape echo as tone color
- NEVER on master bus (except stylistic creative effect)

## HP/LP Filter on Delay Return
Always filter delay returns:
- HP at 200-400Hz: prevent low-end buildup
- LP at 8-12kHz: keep delays from becoming harsh
This makes delays sit naturally without cluttering the mix.
